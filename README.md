## 이미지 자르기 코드입니다.


## 실행:


    WIN 유저: 
    1. npm install
    2. npm install electron-prebuilt
    3. npm start or npm test 
                    
    MAC 유저: 
    1. sudo npm install
    2. sudo npm install electron-prebuilt --save-dev
    3. npm start (with Electron Browser) or npm test (with Browser Sync)
