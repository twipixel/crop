## 이미지 자르기 테스트 코드 입니다.


## 설치, 실행, 빌드:


    WIN & MAC 유저: 
    - npm run install or sudo npm run install
    - npm run crop (Electron 브라우저로 자르기 ES6 버전 실행)
    - npm run hit (Electron 브라우저로 자르기 ES6 충돌 검사 데모 실행)
    
    
    BUILD:
    npm run build 
    (오류 시 한번 더 실행 또는 gulp clean 실행 후 gulp build 실행)
